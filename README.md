===
Event APIs client library for node.js

## Support API
* [ATND](http://api.atnd.org/)
* [connpass](http://connpass.com/about/api/)
* [Zusaar](http://www.zusaar.com/doc/api.html)
* [Doorkeeper](http://www.doorkeeperhq.com/developer/api)
